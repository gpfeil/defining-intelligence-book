This is a repository for the storage for working research documents for my current book project, which aims to define 'intelligence' as it is understdood in practices which shape "AI" as an economic and discursive object. Much of the literature review and quantitative elements of research were accomplished using Anthropic's Claude to locate information, and to make generalizations across carefully (human) selected cases. This continues to be an informative exercise in learning LLMs as both a user and a critic.
